# EciParking
Programa en python de control de parqueadero de zonas de bicicletas

### Prerequisitos

python 3.7
tkinter

librerias de python: sqlite3, os, sys, time


### Instalando

1. descargue el repositorio y ejecute EciParking.py dentro de la carpeta src
